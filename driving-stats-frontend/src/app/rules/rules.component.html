Rules for uploaded files

The uploaded file must be a text file (.txt) with any name and of any size.

The files must be formatted such that

  • Each line is blank/empty or contains a single command followed by its parameters
  • The two allowed commands are Driver and Trip
  • The parameters for each command are space delimited and no extra information should be included after the parameters

For the Driver command, the parameter in order are

  • name: a string value that represent the driver's name

For the Trip command, the parameter in order are

  • driversName: a string value the represents the name of the driver who took this trip
  • startTime: a time value of the form mm::ss that represents the start time of the trip
  • stopTime: a time value of the form mm::ss that represents the stop time of the trip
  • milesDriven: a number value that represents the number of miles driven during the trip

Here's an example file:


Driver Dan

Driver Alex

Driver Bob

Trip Dan 07:15 07:45 17.3

Trip Dan 06:12 06:32 21.8

Trip Alex 12:01 13:16 42.0

Which would produce the following output:

Name Miles Driven Average Mpg
Alex 42 34
Dan 39 47
Bob 0
